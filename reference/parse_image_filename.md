# Parse an image filename into components

Parse an image filename into components

## Usage

``` r
parse_image_filename(filename)
```

## Arguments

- filename:

  Image filename like "01Sep23_NOR_Mid_Mica_Ortho.tif" or
  "01Sep23_NOR_Mid_Mica_Ortho\_\_NDVI.tif" or
  "01Sep23_NOR_Mid_Mica_DEM.tif"

## Value

List with date, site, tide, sensor, layer_type
