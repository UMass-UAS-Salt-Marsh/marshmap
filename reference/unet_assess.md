# Simple wrapper for single-stage assessment (existing behavior)

Simple wrapper for single-stage assessment (existing behavior)

## Usage

``` r
unet_assess(model_path, data_dir, site, dataset = "test")
```

## Arguments

- model_path:

  Path to model

- data_dir:

  Data directory

- site:

  Site code

- dataset:

  Which dataset ('test' or 'validate')
