# Read a predict grid and plot it nicely

Read a predict grid and plot it nicely

## Usage

``` r
plot_predict(
  tiff = "C:/Work/etc/saltmarsh/data/oth/predicted/predict_oth_2025-Apr-28_13-54.tif",
  pal = "ggsci::category20b_d3"
)
```

## Arguments

- tiff:

  A predicted geoTIFF

- pal:

  Selected palette
