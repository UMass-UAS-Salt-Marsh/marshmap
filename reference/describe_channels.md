# Describe channels from image filenames

Describe channels from image filenames

## Usage

``` r
describe_channels(images)
```

## Arguments

- images:

  Character vector of image filenames

## Value

Character string like "26 channels: 2023 summer-mid
(mica,DEM,NDVI,NDRE,NDWIg), post-mid (mica,NDVI,NDRE)"
